# プライバシーポリシー
## お客様から取得する情報
本アプリは、お客様のご利用にあたり、以下の情報をお客様の端末に保存し、当社が管理・利用する外部サーバーには送信・保存しません。
またこれらの情報がお客様及び第三者にとっていかなる重要なものであっても、利用規約でそのような情報を取り扱うことを禁止しているため、本アプリの利用において被った損害については一切責任を負いません。

- OSが生成するID、端末の種類、端末識別子等のお客様が利用するOSや端末に関する情報
- 本アプリの起動時間、入力履歴、購買履歴等のアプリの利用履歴
- プロジェクト名
- タスク名
- ユーザーが記載したタスクに関するメモ
- タスクの開始日時と終了日時
- タスクの進捗状況
- ユーザーが作成したテーマ情報(名と設定画像を含む)

## お客様の情報を利用する目的
本アプリは、お客様から取得した情報を、以下の目的のために利用します。
- お客様の端末内で本アプリの機能を提供・実現するため
- お客様の端末内で本アプリの利用履歴を管理するため
- お客様からのお問い合わせに対応するため
- 本アプリの規約や法令に違反する行為に対応するため
- 本アプリのサービスの変更、提供中止、終了、契約解除をご連絡するため
- 本アプリ規約の変更等を通知するため
- 他、本アプリの機能提供および快適な動作の維持のため


## 安全管理のために講じた措置
【重要：当社による情報の管理と保存場所について】 本アプリが保存する上記の情報は、すべてお客様の端末内に保存されており、当社のサーバーや当社が利用する外部のシステムには送信・保存されません。 そのため、当社は上記のお客様の個人データにアクセスしたり、これを管理したりする権限を持ちません。 
これにより上記の情報に関する安全管理はお客様の端末に依存します。 この性質上、当社は法令に基づく場合を含め、**お客様ご自身のデータ開示や削除の請求に対応することは物理的にできません。**
データの管理（入力、バックアップ、削除等）はお客様ご自身の責任で行っていただく必要があります。ユーザーが任意でiCloud等の外部サービスと連携して行うバックアップについては、当該サービスのプライバシーポリシーが適用されます。

## 第三者提供
上記の通り、本アプリが保存する情報はいずれもお客様の端末内のみに存在するため、当社から第三者へ提供することはありません。
但し、次の場合は除きます。
- 個人データの取扱い及び仕様を外部に委託する場合
- 本アプリが買収され、個人データの取扱い及び仕様が変更された場合
- 事業パートナーと共同利用する場合（具体的な共同利用がある場合は、その内容を別途公表します。）
- その他、法律によって合法的に第三者提供が許されている場合
- Apple社が提供する機能によりデータが取り扱われた場合

## プライバシーポリシーの変更
当社は、必要に応じて、このプライバシーポリシーの内容を変更します。この場合、変更後のプライバシーポリシーの施行時期と内容を適切な方法により周知または通知します。
